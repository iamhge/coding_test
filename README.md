## 2. 문제 사이트
* 백준(boj)
  * 네이밍 규칙 : 앞에 'boj_'가 붙고, 뒤에 문제 번호를 표기한다.
  * 문제의 최상단에 문제 제목과 함께, 문제 풀이에 사용된 아이디어, 참고 링크 등이 기재되어있다.
  * 문제 풀이 후, 다른 사람들의 코드를 참고한 경우 '# 다른 사람 코드' 라고 주석을 단 후, 뒤에 해당 코드를 삽입했다.
